# 用户操作手册

本文档主要面向系统管理员和内容运营人员，介绍如何使用后台管理系统管理学员作品。

## 1. 登录后台

访问以下地址进入管理后台：
`http://yourdomain.com/admin/`

使用管理员账号和密码登录。

## 2. 作品管理

在后台首页，找到 **Showcase** 应用下的 **学员作品** (Works) 链接，点击进入列表页。

### 2.1 添加新作品

点击右上角的 **增加 学员作品 +** 按钮。

#### 填写字段说明

| 字段名       | 说明               | 必填   | 备注                                                   |
| :----------- | :----------------- | :----- | :----------------------------------------------------- |
| **作品标题** | 作品的主标题       | 是     | 简明扼要，建议 10-20 字                                |
| **分类**     | 作品所属的大类     | 是     | 选项：AI多模态创作、AI编程应用、AI数字人               |
| **作品类型** | 决定展示方式       | 是     | 图片：展示大图；视频：播放器；网页链接：跳转按钮       |
| **学员姓名** | 作者姓名           | 是     |                                                        |
| **作品描述** | 详细介绍           | 否     | 支持多行文本，将在详情弹窗中显示                       |
| **封面图片** | 列表页显示的缩略图 | **是** | 建议比例 4:3 或 16:9，JPG/PNG 格式                     |
| **媒体文件** | 详情页展示的文件   | 否     | 当类型为图片/视频时使用                                |
| **网页链接** | 跳转的目标 URL     | 否     | 当类型为网页链接时填写，需包含 `http://` 或 `https://` |
| **是否发布** | 控制前台可见性     | 是     | 默认勾选。取消勾选后前台不可见                         |
| **排序权重** | 控制显示顺序       | 是     | 数字**越大**越靠前显示。默认 0                         |

### 2.2 编辑作品

在作品列表中点击任意作品的 **标题** 即可进入编辑页面。修改完成后点击底部的 **保存** 按钮。

### 2.3 删除作品

- **单个删除**: 进入编辑页面，点击左下角的 **删除** 按钮。
- **批量删除**: 在列表页勾选多个作品，在顶部的 "动作" 下拉框中选择 "删除所选的 学员作品"，然后点击 **执行**。

## 3. 常见问题 (FAQ)

### Q: 上传的图片在前台不显示？
A: 请检查是否填写了 **封面图片**。如果是详情页媒体文件不显示，请确保上传了对应的 **媒体文件** 并且当前作品类型选择正确。

### Q: 视频文件支持什么格式？
A: 建议使用 **MP4** 格式（H.264 编码），以确保在所有浏览器中都能正常播放。

### Q: 如何调整作品在首页的顺序？
A: 修改作品的 **排序权重** 字段。权重数值高的排在前面。如果权重相同，则按创建时间倒序排列（最新的在前面）。
